# Post: Protestas de la "Generación Z" reclamando seguridad
*Autor: Gianfranco Romero*

## Descripción
Este post busca analizar los patrones de protestas que tienen como reclamo la seguridad ciudadana. Se hace en el contexto de las marchas de la Generación Z en el Perú, que se enmarca en un fenómeno global de [protestas de la Generación Z](https://es.wikipedia.org/wiki/Protestas_de_la_generaci%C3%B3n_Z) que se propagaron en diferentes países. En el contexto peruano, estas protestas tuvieron a la seguridad ciudadana como [una de sus principales demandas](https://www.bbc.com/mundo/articles/cj4y0y7kekwo).

## Datos usados
Este post usa datos de la *Base de Eventos de Protestas en el Perú (1980 - 2024)*, ofrecidos por la Escuela de Gobierno y Políticas Públicas de la Pontificia Universidad Católica del Perú. Los datos son de acceso abierto, y pueden ser solicitados mediante [esta página web](https://democracia-gobernabilidad.pucp.edu.pe/participaci%C3%B3n/protestas-sociales/bases-de-datos).

## Archivos incluídos
En este repositorio se incluyen:
1. La Base de Eventos de Protestas en el Perú (1980 - 2024), de la PUCP
2. Un libro de códigos para la base de datos
3. Un manual de usuario para la base de datos
4. El código, en formato R script, con el que se generaron los gráficos del post
5. Los gráficos usados en el post, en formato .svg
